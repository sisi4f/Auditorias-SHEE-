⚙ Apps Script — sincronização gratuita

Sem API Key, sem cartão de crédito. Um único Apps Script sincroniza todas as farmácias — cada uma fica numa aba separada da mesma Google Sheet.

Passo 1 · Criar a Google Sheet
  1. Vai a sheets.google.com e cria uma Sheet nova
  2. Dá o nome que quiseres (ex: "Auditorias Farmácias 2026")
  3. Não precisas de criar cabeçalhos — o script cria as abas automaticamente
Passo 2 · Apps Script
  1. Na Sheet: Extensões → Apps Script
  2. Apaga o código existente e cola o código abaixo
  3. Guardar → Implementar → Nova implementação
  4. Tipo: Aplicação Web · Executar como: Eu · Acesso: Qualquer pessoa
  5. Copia o URL gerado
const HEADERS = ['id','area','name','obs','status','prio','resp','start','end','cost','prog','photos','lastEdit','editBy','created','notes']; function doGet(e) { const sheetName = e.parameter.sheet || 'Sheet1'; const ss = SpreadsheetApp.getActiveSpreadsheet(); let sh = ss.getSheetByName(sheetName); if (!sh) { // Return list of all sheet names for overview const names = ss.getSheets().map(s => s.getName()); return json({ ok: true, sheets: names, rows: [] }); } const data = sh.getDataRange().getValues(); if (data.length < 2) return json({ ok: true, rows: [] }); const rows = data.slice(1).map(r => { const obj = {}; HEADERS.forEach((h, i) => obj[h] = r[i] !== undefined ? String(r[i]) : ''); return obj; }); return json({ ok: true, rows }); } function doPost(e) { try { const p = JSON.parse(e.postData.contents); const ss = SpreadsheetApp.getActiveSpreadsheet(); const sheetName = p.sheet || 'Sheet1'; let sh = ss.getSheetByName(sheetName); if (!sh) sh = ss.insertSheet(sheetName); if (p.action === 'init') { sh.clearContents(); sh.getRange(1, 1, 1, HEADERS.length).setValues([HEADERS]); sh.getRange(1, 1, 1, HEADERS.length).setFontWeight('bold'); } if (p.action === 'write' && p.rows) { const last = sh.getLastRow(); if (last > 1) sh.deleteRows(2, last - 1); if (p.rows.length > 0) sh.getRange(2, 1, p.rows.length, p.rows[0].length).setValues(p.rows); } if (p.action === 'delete') { ss.deleteSheet(sh); } return json({ ok: true }); } catch(err) { return json({ ok: false, error: err.message }); } } function json(obj) { return ContentService.createTextOutput(JSON.stringify(obj)) .setMimeType(ContentService.MimeType.JSON); }

🛡 Integridade de Dados

O app guarda automaticamente backups locais rotativos (últimas 3 versões por farmácia, a cada 5 minutos de edição).
A Google Sheet funciona como cópia de segurança remota permanente — mesmo que percas o browser local, os dados estão na Sheet.

Garantias de colaboração:
· Merge inteligente por lastEdit — ganha sempre a versão mais recente, campo a campo
· Notificação visual quando um colega atualizou tarefas desde a última sincronização
· Campo editBy regista quem fez cada alteração
· Sincronização automática a cada 60 segundos quando o Apps Script está configurado
